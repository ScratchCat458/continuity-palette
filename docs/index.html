Vibrant for a purpose...

Continuity is a palette designed to be a vibrant colour set for syntax highlighting and presentation purposes. This inspiration comes from the colour choices of Freya Holm√©r, though I didn't end up sampling any of them during the creation process.

Continuity is composed of two systems, the first used for general apllications with all values fully defined and the other being an extension of this to support all tones.

Standard Palette

The standard palette conists of 8 accents, a background and foreground colour and 2 shades of white for text content. Primarily, the accents that should be used are purple, cyan, green and pink. The other accent colours exist to complete the set, with blue using the same hue as the background colour.

The accent colours all have a saturation of 80% which makes them pop against the darker background. White is the standard colour which should be used for text due to the contrast, however the variant can be used for text that should be less visible, such as the links at the top of this page. Foreground should only be used on top of or adjacent to Background, rather than in isolation.

Implementations

So far the only implementations are stylesheets for web applications. In future, the plan is to intergrate support for art software, syntax highlighting theme and other UI platforms.

If one of these isn't implemented yet or there is a platform that you would like support for, write it yourself and file a pull request.

  • Art Software
